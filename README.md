# Andromeda Mod

Andromeda.Mod is the runtime gameplay mod for Enemy On Board.

This repository includes CI for build validation and a release workflow that publishes Andromeda.Mod.dll and Andromeda.Mod.zip as GitHub release assets. The installer consumes those release assets from the latest Andromeda.Mod release.

To run CI/CD builds in GitHub Actions, configure the repository secret ANDROMEDA_GAME_MANAGED_ZIP_URL to a zip that contains the Enemy On Board managed DLL set, including Assembly-CSharp-Publicized.dll.
